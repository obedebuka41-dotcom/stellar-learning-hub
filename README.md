# Stellar Learning Hub

An interactive learning platform for Stellar blockchain development with tutorials, guides, and hands-on examples.

## Features

- **Stellar Basics**: Introduction to accounts, transactions, and the Horizon API
- **Smart Contracts**: Soroban/Rust contract development tutorials
- **dApp Development**: Frontend integration with Freighter wallet and backend SDKs

## Tech Stack

- Next.js 15
- TypeScript
- Tailwind CSS

## Getting Started

```bash
npm install
npm run dev
```

Open http://localhost:3000 to view the application.

## Project Goals

- Create interactive learning modules for Stellar concepts
- Build real-world example applications
- Document best practices for Stellar development
- Foster community contributions
